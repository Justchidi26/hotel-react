## Setup Steps

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone <https://github.com/Justchidi26/hotel-react.git>
````

2. **Navigate into the project folder**

   ```bash
   cd <your-project-folder>
   ```

3. **Install dependencies**

   ```bash
   npm install
   ```

4. **Start the development server**

   ```bash
   npm run dev
   ```

5. **Open in browser**

   ```
   http://localhost:3000
   ```

   or the port shown in your terminal

## Components Created

The project is structured using reusable React components:

* `<Navbar />` – Top navigation bar
* `<Hero />` – Hero section with background image
* `<Search />` – Search functionality overlay on hero section
* `<Destinations />` – Displays destination categories
* `<Hotel />` – Hotel listings section
* `<Cities />` – Popular cities section
* `<States />` – State-based listings
* `<Deals />` – Special offers and deals
* `<Features />` – Platform features section
* `<HotelApp />` – App promotion section
* `<EmailForm />` – Newsletter/email subscription form
* `<Footer />` – Footer with links and information

## Known Limitations

* ❌ No backend integration (static UI only)
* ❌ Search component is not fully functional (UI only)
* ❌ Some Links are not connected to real routes/pages
* ❌ Data is hardcoded (no API or dynamic content)
* ❌ Limited accessibility optimizations
* ❌ Images and content are not optimized for performance
